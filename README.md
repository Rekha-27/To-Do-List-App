# To-Do List Web App

A simple, responsive task manager built using ReactJS and TailwindCSS.

## 🛠️ Tech Stack
- Frontend: ReactJS
- Styling: TailwindCSS
- Storage: LocalStorage

## ✨ Features
- Add, delete, and complete tasks
- Real-time UI updates with React Hooks
- Persistent task list using LocalStorage
